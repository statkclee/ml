기계와의 경쟁을 준비하며…

“The future is here, it’s just not evenly distributed yet.”

학습 목차

  1. 환경설정
    1. 명령라인 데이터 분석 맛보기
      1. 대용량 데이터 표본추출
    2. 파이썬기반 기계학습 툴체인(toolchain): 파이썬 기계학습 생태계
    3. 기계학습 (순수한) 맛보기: 전통적인 통계모형 개발과정 맛보기
  2. 기계학습 지도
    1. 3가지 기계학습 원리
      1. (사업) 문제의 정의
      2. 데이터(Data)
      3. 데이터를 통한 문제풀이 과정
    2. 기계학습 알고리즘
      1. 활용도 높은 기계학습 알고리즘 – 분류, 예측, 군집화
  3. 기계학습 적용
    1. 테스트 주도 기계학습
    2. 알고리즘 성능평가
    3. 데이터 전처리
      1. 예측모형 적용을 위한 전처리
      2. 데이터 전처리 및 정제(파이썬)
    4. 탐색적 데이터 분석(EDA)
    5. 데이터에 무게를 둔 예측모형
      1. 인간 중심 기계학습 모형
        1. 선형대수와 함께하는 회귀분석 이해
        2. 통계적 모형개발 기초
        3. 전통방식 모형개발 -타이타닉 생존데이터
      2. 인간과 기계가 함께하는 기계학습 예측모형
        1. 기계학습 예측모형 준비
        2. 기계학습 예측모형 실습
        3. 모형식별 및 선택
        4. 사례 - 도요타 중고차 가격 예측
        5. 사례 - 콘크리트 강도
        6. 사례 - 카드발급
      3. 기계 중심 기계학습
  4. 데이터 유형별 기계학습 모형
    1. 네트워크 데이터 들어가며 – 색상과 글꼴
      1. 네트워크 데이터
      2. 정적 네트워크 데이터 시각화
      3. 동적 네트워크 데이터 시각화
      4. 기타 네트워크 데이터 분석
      5. R 팩키지 사회망 분석
      6. 네트워크 데이터 연습문제
      7. 네트워크 분석 사용자 안내서
      8. 네트워크 유형
      9. 네트워크 커뮤니티 탐지
    2. 텍스트 데이터
      1. 텍스트 데이터 수집 – 트위터
      2. 텍스트 데이터 전처리 – qdap
      3. 단어문서행렬 – 단어 빈도
      4. 소설 텍스트 데이터 분석 – 소나기
    3. 장바구니 데이터 분석
    4. 이상점(Outlier) 검출
  5. 기계학습 상용화
    1. 신용평점모형 개발
      1. 신용평점모형 탐색적 데이터 분석, 전처리
      2. 신용평점모형 - 로지스틱 회귀모형과 의사결정나무
      3. 신용평가와 신용평점모형
      4. 소매금융 부도예측 - 독일신용 데이터
    2. 랜딩클럽 - 채무불이행 예측모형
    3. 예측모형 활용분야
    4. 마케팅 반응 예측모형
  6. 기계학습 운영 및 자동화 - 파이프라인
