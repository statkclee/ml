학습 목차

  1. 환경설정
    1. 명령라인 데이터 분석 맛보기
      1. 대용량 데이터 표본추출
    2. 파이썬기반 기계학습 툴체인(toolchain): 파이썬 기계학습 생태계
    3. 기계학습 (순수한) 맛보기: 전통적인 통계모형 개발과정 맛보기
  2. 기계학습 지도
    1. 3가지 기계학습 원리
      1. (사업) 문제의 정의
      2. 데이터(Data)
      3. 데이터를 통한 문제풀이 과정
    2. 테스트 주도 기계학습
    3. 기계학습 알고리즘
      1. 나무기반 기계학습모형
      2. 기계학습 예측모형 준비
    4. 데이터 전처리
      1. 예측모형 적용을 위한 전처리
      2. 데이터 전처리 및 정제(파이썬)
    5. 탐색적 데이터 분석(EDA)
      1. R 조합수학(Combinatorics)
      2. 범주형 변수와 표
      3. 텍스트 콘솔기반 탐색적 데이터 분석(EDA)
      4. 이상점(Outlier) 검출
      5. 이상점 검출 - mahalanobis
      6. 비지도 학습 - 포켓몬
      7. 군집분석(Clustering) - 포켓몬
    6. 알고리즘 성능평가
  3. 기계학습 운영 및 자동화 - 파이프라인
    • 기계학습 클라우드(AWS) 개발배포 환경
      1. 시나리오별 클라우드(AWS) 컴퓨팅 자원을 데이터 과학에 활용
      2. AWS 우분투 EC2 + S3 버킷 + RStudio 서버
      3. 단독형 스파크 설치 - PC/노트북, EC2 원격 컴퓨터
      4. 스파크 EC2 클러스터 - 부싯돌(flintrock)
        1. 스파크 EC2 클러스터 - 데이터과학 툴체인(R, sparklyr)
      5. EMR 스파크 클러스터 - wadal
      6. 분석할 데이터를 스파크에 적재하는 방법 - S3 포함
      7. dplyr을 Spark 위에 올린 sparklyr
    • 예측모형 개발 자동화
      • R 모형설계행렬(Recipe)
      • R기반 빅데이터 개발/운영환경
      • R 모형개발 자동화(recipe, caret, twidlr)
    1. 데이터에 무게를 둔 예측모형
      1. 인간 중심 기계학습 모형
        1. 선형대수와 함께하는 회귀분석 이해
        2. 통계적 모형개발 기초
        3. 전통방식 모형개발 -타이타닉 생존데이터
      2. 인간과 기계가 함께하는 기계학습 예측모형
        1. 기계학습 예측모형 실습
        2. 모형식별 및 선택
        3. 회귀모형 → caret → 스파크 → \(H_2 O\) - 1부
        4. 회귀모형 → caret → 스파크 → \(H_2 O\) - 2부
        5. 사례 - 도요타 중고차 가격 예측
        6. 사례 - 콘크리트 강도
        7. 사례 - 카드발급
        8. 사례 - 기계고장 빈도수 예측
      3. 기계 중심 기계학습
  4. 데이터 유형별 기계학습 모형
    1. 네트워크 데이터
      • 보다 심화된 네트워크 데이터에 대한 분석을 위해 별도 저장소로 이동.
    2. 텍스트 데이터와 저작
      1. 텍스트 데이터 다루기
        1. 숫자를 문자로 표현
        2. stringr을 통해 문자열/텍스트 다루기
        3. 해외연극 등장인물 출현횟수
        4. 국내영화(건축학 개론) 등장인물 출현횟수
      2. 텍스트 마이닝(text mining)
        1. 텍스트 단어구름(Word Cloud)
        2. 텍스트 데이터 수집 – 트위터
        3. 단어문서행렬 – 단어 빈도
        4. 소설 텍스트 데이터 분석 – 소나기
        5. 저녁이 있는 삶 – 손학규
        6. 연설문 판별 - 오바마 vs. 롬니
      3. 평점 텍스트 데이터
        1. 텍스트 분류(Text Classification) - 나이브 베이즈(naive bayes)
        2. 고객 방문후기 평점 감성분석 - 옐프(Yelp)
        3. 영화 평점 - 무비렌즈(MovieLens)
      4. 감성분석(Sentiment Analysis) - 깔끔한 텍스트 방식(tidytext)
      5. 정규표현식에서 워드2벡(Word2Vec)
    3. 장바구니 데이터 분석
  5. 기계학습 상용화
    1. 기계학습 - 기술부채(Technical Debt)
    2. 신용평점모형 개발
      1. 신용평점모형 탐색적 데이터 분석, 전처리
      2. 신용평점모형 - 로지스틱 회귀모형과 의사결정나무
      3. 신용평가와 신용평점모형
      4. 소매금융 부도예측 - 독일신용 데이터
    3. 랜딩클럽 - 채무불이행 예측모형
    4. 마케팅 반응 예측모형
    5. 고객 이탈(churn) 모형
    6. 마케팅 - RFM 및 응용분야
