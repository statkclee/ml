학습 목차

  1. 환경설정
    1. 기계학습 (순수한) 맛보기: 전통적인 통계모형 개발과정 맛보기
    2. 기계학습 (고객용) 맛보기: 전통적인 통계모형에 상업적 요소를 가미
    3. 명령라인 데이터 분석 맛보기
    4. 파이썬기반 기계학습 툴체인(toolchain): 파이썬 기계학습 생태계
  2. 기계학습 지도
    1. 3가지 기계학습 원리
      1. 문제의 정의
      2. 데이터(Data)
      3. 데이터를 통한 문제풀이 과정
    2. 알고리즘
  3. 기계학습 적용
    1. 테스트 주도 기계학습
    2. 알고리즘 성능평가
    3. 데이터 전처리 및 정제
    4. 데이터 적합 모형 개발
      1. 예측모형 A-Z(연속형)
      2. 예측모형 A-Z(연속형) - 콘크리트 강도
      3. 예측모형 A-Z(이산형)
      4. 모형식별 및 선택
      5. 문서 분류
    5. 네트워크 데이터
      1. R 팩키지 사회망 분석
    6. 소설 텍스트 데이터 분석 기초
    7. 태그 혹은 단어 구름(tag or word cloud)
  4. 기계학습 상용화
    1. 기계학습 적용
